Introduction to JavaScript

JavaScript ― это интерпретируемый язык программирования общего назначения с динамической типизацией данныхю

Интерпретируемый значит что программа выполняется в исходном виде, т.е., исходный код не преобразуется в машинный или промежуточный код. В связи с этим ошибок на этапе компляции и компановки отсутствуют, есть только ошибки на этапе выполнения. Итнерпретатор JavaScript может быть установлен в любую операционную систему как семейство Windows, так и семейство Unix. Это дает хорошую переносимость кода или кроссплатформенность. JavaScript также иногда используют для разработки мобильных приложений под Android.JavaScript также часто используют в тестировании для написания скриптов.Но не смотря на все это чаще всего JavaScript используется для разработки Web приложений, а именно FrontEnd части Web приложения.Основным и самым распространенным интерпретатором JavaScript является самый обычный браузер.

Динамическая типизация говорит о том, что тип переменных и констант определяется во время присваивания по типу значения справа (r-value); l-value = r-value; Это в свою очередь означает то, что у переменной нет постоянного типа данных, например, в переменную в которой хранится число, можно записать строку и во время записи она изменит свой тип, это достигается благодаря тому, что абсолютно все типы являются объектами, а также благодаря полиморфизму.

Переменные и типы данных

Переменные и константы

var
Объявляет глобальную переменную, несмотря на область видимости, в которой она объявлена. Если записать значение в несуществующую переменную, то это не вызовет ошибки, то это будет аналогично ключевому слову 'var';
let
Объявляет локальную переменную, только для той области видимости, в которой она используется;
const
Объявляет константу. Константы могут быть только локальными;

Типы данных:

Делятся на примитивные типы и объекты

  1. Примитивные типы:
    • Boolean ― принимает одно из двух значений Trueлибо False. Значения: 0, null, '', "",`` неявно преобразуется вFalse. True ― это любое значение, отличное от нуля или пустой строки.
    • String ― объект, описывающий массив символов;
    • Number ― вещественное(дробное число), которое является полным аналогом типа double. Все числовые литералы неявно преобразуются в Number;
    • Bigint ― целое число неограниченных размеров;
  2. Объекты:
    • Object
    • Array
    • Function
    • Date
    • Map
    • Set
    • Regexp― регулярное выражение.
